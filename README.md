## About

Sledo is a tool to automatically generate connected demo data.

## Setup

```bash
git clone https://github.com/luxbe/sledo
cd sledo
pipenv install
```

## 💻 Develop

## 📥 Install dependencies

```bash
pipenv install
```

## 🛠️ Run sledo

```bash
pipenv run python -m sledo
```

## 🔬 Run tests

```bash
python -m unittest discover tests
```
